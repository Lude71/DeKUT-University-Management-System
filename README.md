# DeKUT University Management Database System

## Project Overview

This project is a relational database management system developed using SQL Server Management Studio (SSMS). The system simulates a university management database for Dedan Kimathi University of Technology (DeKUT).

The project demonstrates fundamental and intermediate SQL concepts including:

* Database creation
* Table creation
* Primary keys
* Foreign keys
* Data insertion
* Data querying
* INNER JOIN operations
* Aggregate functions
* GROUP BY and HAVING clauses
* Filtering and sorting data

## Tables Included

1. Students
2. Library
3. Finance
4. Classrooms
5. Vehicles
6. Staff

## Technologies Used

* SQL Server Management Studio (SSMS)
* Microsoft SQL Server

## Skills Demonstrated

* Relational database design
* Data analysis using SQL
* Query optimization basics
* Database relationships
* Structured Query Language (SQL)

## Example Features

* Track student fee balances
* Manage library books
* Store staff information
* Manage classrooms and vehicles
* Generate reports using SQL queries
